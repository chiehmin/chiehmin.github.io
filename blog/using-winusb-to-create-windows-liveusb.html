Using WinUSB to create Windows liveusb

On 2016/10/29 at 12:09

I want to install Windows 10 to another disk of my PC. However, I don't have a Windows OS availabe to create the Windows liveusb. I came accross to a wonderful opensource project called WinUSB. It can create windows liveusb under linux environment.

I made a change to make it runnable under Fedora 24 environment. I changed the script to use grub2-install if grub-install is not detected. I submitted a pull request and wait for it to be accepted. Here's my commit.

Usage instructions: 

# Installing wxWidget dependency
$ sudo dnf install wxGTK3-devel
# Pluging-in my usb disk and umount it (It is automatically mounted by my Fedora)
$ sudo umount /dev/sdc1
# Formating the disk and installing win10.iso to the usb disk. It works like a magic!
$ sudo /usr/local/bin/winusb --format ./win10.iso /dev/sdc
